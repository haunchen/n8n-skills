Features

📚

Comprehensive Node Knowledge Base

Detailed documentation for 545 n8n nodes, organized into 6 main categories, supports the latest n8n v2.13.4

🤖

Multi-Platform AI Integration

Supports Claude Code, Claude.ai Web, and Claude Desktop through seamless Skill Pack format integration

⚡

Priority Ranking

Multi-dimensional scoring system based on usage frequency, documentation completeness, community popularity, and versatility

🔄

Regular Release Updates

Automatically tracks the latest n8n versions weekly and publishes updates, download to get the latest content

📁

Complete Resource Structure

Main SKILL.md provides quick reference, resources/ directory contains detailed documentation and templates organized by category
